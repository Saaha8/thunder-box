# ThunderCore Truffle Box

## Installation

1. Install Truffle globally.

```bash
npm install -g truffle
```

2. Download the box.

```bash
truffle unbox thunder-box
```

3. Run the development console.

```bash
truffle develop
```

4. Compile and migrate the smart contracts.

```bash
compile
migrate
```
